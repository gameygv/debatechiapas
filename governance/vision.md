# Solution Vision: debatechiapas

## Identity

### Description

Debate Chiapas es una plataforma de publicación digital enfocada en debate, opinión y noticias desde Chiapas, México. Facilita la creación, gestión y distribución de contenido periodístico con asistencia de IA (generación, reescritura, clasificación automática). Fork del CMS MoyMontes, adaptado para un nuevo medio independiente.

## Outcomes

| **Outcome** | **Description** |
|-------------|-----------------|
| **Publicación ágil** | Periodistas y editores publican artículos en minutos asistidos por IA para generación, reescritura y clasificación |
| **Alcance social** | Distribución automática a redes sociales con OG tags optimizados y SSR para crawlers |
| **Gestión editorial** | Panel admin con roles (superuser/editor), biblioteca de medios, secciones, tags y banners |
| **Monetización** | Sistema de anuncios con métricas (vistas/clicks) y banners programables por fecha |
| **Autonomía técnica** | Supabase self-hosted, sin dependencia de servicios cloud propietarios para datos |
